# dotfiles

> dotfiles for @ragingwind

# How to start

Bootstrap will install an enssential programs. For example rvm, nvm and brew, and then try to configure and copy dotfiles into your root

```sh
sh ./bootstrap.sh

# npm start:[TARGETOS]
npm start:macos

# or

npm start:ubuntu
```
